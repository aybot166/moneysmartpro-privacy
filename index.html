MoneySmartPro 隐私政策

生效日期：2026 年 6 月 21 日 v1.0

📋 引言

MoneySmartPro（以下简称「我们」或「本应用」）深知个人隐私的重要性。 本隐私政策旨在向您清晰说明本应用如何对待您的数据。

核心承诺：本应用不收集、不上传、不分享您的任何个人数据。 所有数据仅保存在您的设备本地。

💾 数据收集与存储

本应用 不会 向任何远程服务器发送您的数据。 您的所有数据仅存储在设备本地：

  • 应用数据 — 您的记账记录、分类设置、预算等核心数据全部保存在 iOS 本地存储中， 由 Apple Core Data 框架管理。
  • API 密钥 — 您自行提供的 DeepSeek API Key 仅存储在 iOS 钥匙串（Keychain）中， 受系统级加密保护。

我们无法访问您的任何数据。 本应用没有后端服务器，不提供云端同步功能，不会将任何信息传输到开发者控制的任何服务器。

🔗 第三方服务

DeepSeek API

本应用允许您自行配置 DeepSeek API Key， 以实现 AI 智能记账分析功能。当您启用此功能时：

  • 您的 API Key 仅保存在设备本地钥匙串中
  • AI 请求直接从您的设备发送到 DeepSeek API 服务器
  • 我们不会经手、中转或记录您的任何 API 请求内容

使用 DeepSeek API 即表示您同意 DeepSeek 的服务条款： platform.deepseek.com/terms

如您不提供 API Key，本应用所有功能均可正常离线使用，不受任何影响。

🛒 内购与订阅

本应用通过 Apple App Store 内购（In-App Purchase）提供高级功能订阅。 所有支付流程由 Apple 处理，适用 Apple 的标准最终用户许可协议： Apple 标准 EULA

  • 我们不收集、不存储您的任何支付信息
  • 所有交易由 Apple 独立处理，我们仅收到订阅状态确认
  • 您可在 iOS 系统设置中随时管理或取消订阅

🔒 数据安全

您的数据安全是我们的首要考虑。本应用的数据保护机制包括：

  • 全本地存储 — 所有数据仅保存在您的设备上
  • 无服务器架构 — 应用没有任何后端服务器，杜绝数据泄露风险
  • 无云同步 — 数据不会自动上传到 iCloud 或任何云服务
  • Keychain 加密 — API Key 使用 iOS 系统级钥匙串加密存储
  • 无第三方分析 — 本应用不集成任何数据分析、广告或追踪 SDK

由于所有数据均在本地，数据的安全性取决于您设备本身的安全性。 我们建议您为设备设置强密码并启用生物识别保护。

👤 用户权利

您对自己的数据拥有完全控制权：

  • 查看数据 — 所有数据随时可在 App 内查看
  • 导出数据 — 可通过 App 内功能导出您的数据
  • 删除数据 — 可在 App 内清除数据，或直接删除 App
  • 一键清除 — 删除 App 即永久删除所有本地数据，无需额外操作

由于我们不持有您的任何数据，因此也不存在传统意义上的「数据删除请求」—— 您只需删除 App，所有数据即被彻底移除。

🧒 儿童隐私

本应用不针对 13 岁以下儿童设计，也不会故意收集儿童的个人信息。 由于本应用不在服务器端收集任何数据，不存在儿童数据泄露的风险。

📝 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布， 并在 App 内通知用户。建议您定期查阅本页面以了解最新信息。
